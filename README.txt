# Fruit Freshness Detection System

## Dataset Folder

Create folder:

App/

Inside App folder:

- AppleFresh
- AppleSpoiled
- BananaFresh
- BananaSpoiled

Put images inside each folder.

---------------------------------------------------

## Install

pip install -r requirements.txt

---------------------------------------------------

## Train Model

python train_yolov5_fruit.py

---------------------------------------------------

## Run Streamlit App

streamlit run app.py

---------------------------------------------------

## Recommended

- 500+ images per class
- Good lighting
- Different backgrounds
- Use GPU for training
